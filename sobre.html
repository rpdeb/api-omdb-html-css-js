Objetivo

Esse site é o lugar para encontrar dados sobre diversas produções do cinema, podendo elas serem filmes ou series.

Desenvolvimento

Esse site foi desenvolvido como pé requisito para obtenção da nota do segundo bimestre, na disciplina : Desenvolvimento de Sistemas para internet.
Alunas: Debora Ribeiro e Ana Luíza Alves.
Terceira série do ensino médio, integrado ao curso de informática.
Desenvolvimento com a supervisão do professor Marcos Dias.
Instituto Federal Tocantins (Campus Dianópolis).

Utilização da api (application programming interface), Open Movie Database (Omdb) e do fetch para fazer as requisições no banco de dados da api.
